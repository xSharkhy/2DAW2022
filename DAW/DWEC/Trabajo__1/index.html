Navigator
×

El objeto JavaScript navigator se utiliza para la detección del navegador. Se puede utilizar para obtener información del navegador como appName, appCodeName, userAgent, etc. El objeto navigator es objeto de la clase Window, por lo que se puede acceder por:

window.navigator

Propiedades

No. Propiedad Descripción
1 appName devuelve el nombre
2 appVersion devuelve la versión
3 appCodeName devuelve el nombre en código
4 cookieEnabled devuelve true si las cookies están activas
5 userAgent devuelve el user agent
6 language devuelve el lenguaje. Sólo soportado en Netscape y Firefox.
7 userLanguage devuelve el lenguaje usuario. Sólo sopoprtado en IE
8 plugins devuelve los plugins. Sólo soportado en Netscape y Firefox.
9 systemLanguage rdevuelve el lenguaje del sistema. Sólo soportado en IE.
10 mimeTypes[ ] devuelve un array de mime. Sólo soportado en Netscape y Firefox.
11 platform devuelve la plataforma.
12 online devuelve true si el navegador está conectado

Métodos

No. Método Descripción
1 javaEnabled() comprueba si java está habilitado.
2 taintEnabled() comprueba si taint está habilitado. Obsoleto desde JavaScript 1.2.

Screen
×

El objeto JavaScript screen contiene información de la pantalla del navegador. Se puede utilizar para mostrar el ancho de la pantalla, la altura, colorDepth, pixelDepth, etc. El objeto navigator es objeto de la clase Window, por lo que se puede acceder por:

window.screen

Propiedades

No. Propiedad Descripción
1 width devuelve el ancho de la pantalla
2 height devuelve el alto de la pantalla
3 availWidth devuelve el ancho disponible
4 availHeight devuelve el alto disponible
5 colorDepth devuelve la profundidad de color
6 pixelDepth devuelve la profundidad de píxel

Window
×

El objeto ventana representa una ventana en el navegador. Un objeto de ventana es creado automáticamente por el navegador. Window es el objeto del navegador, no es el objeto de javascript. Los objetos de javascript son string, array, date, etc.

Métodos

Método Descripción
alert() muestra una ventana de alerta con un botón de OK
confirm() muestra una ventana de confirmación con botones de OK y cancelar
prompt() displays a dialog box to get input from the user.
open() opens the new window.
close() closes the current window.
setTimeout() performs action after specified time like calling function, evaluating expressions etc.

History
×

El objeto historial de JavaScript representa una matriz de URLs visitadas por el usuario. Utilizando este objeto, se puede cargar la página anterior, la anterior o cualquier página en particular. Historial es objeto de la clase Window, por lo que se puede acceder a él por:

window.history

Propiedades

Property Description
length returns the number of URLs in the history list.

Métodos

Method Description
back() loads the previous URL in the history list.
forward() loads the next URL in the history list.
go() loads a specific URL from the history list.

Document
×

El objeto documento representa todo el documento html. Cuando el documento html se carga en el navegador, se convierte en un objeto documento. Es el elemento raíz que representa el documento html. Tiene propiedades y métodos. Con la ayuda del objeto documento, podemos añadir contenido dinámico a nuestra página web. Como se mencionó anteriormente, es objeto de la clase Window. Así que esta es la forma de llamarlo:

window.document
According to W3C - "The W3C Document Object Model (DOM) is a platform and language-neutral interface that allows programs and scripts to dynamically access and update the content, structure, and style of a document."

Métodos

Method Description
write("string") writes the given string on the doucment.
writeln("string") writes the given string on the doucment with newline character at the end.
getElementById() returns the element having the given id value.
getElementsByName() returns all the elements having the given name value.
getElementsByTagName() returns all the elements having the given tag name.
getElementsByClassName() returns all the elements having the given class name.

Location
×

El objeto Location representa la URL de la página actual. Con la ayuda de este objeto, podemos obtener la URL actual, redirigir a otra URL, obtener la URL anterior, obtener la URL anterior, etc. El objeto Location es un objeto de la clase Window. Así que se puede acceder a él por:

window.location

Propiedades

Property Description
href returns the URL of the current page.
protocol returns the web protocol used (http: or https:).
host returns the web host (hostname:port).
hostname returns the web hostname.
port returns the web port (80 or 443).
pathname returns the path and filename of the current page.
search returns the query string of the current page.
hash returns the anchor name (portion of the URL after #).

Métodos

Method Description
assign() loads a new document.
reload() reloads the current document.
replace() replaces the current document with a new one.
